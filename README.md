# LearnTheme.in - Online Nursing Study Platform

A professional mobile-first online learning web application designed for Nursing students to access notes, attend MCQ quizzes, and track courses.

## 🚀 Features
- 📱 **Mobile-First UI:** Looks and behaves like a native mobile app.
- 🗺️ **Bottom Navigation:** Quick access to Home, My Courses, MCQ, and Profile.
- ⚡ **Supabase Backend:** Scalable database & secure file storage.

## 📁 Tech Stack
- **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript
- **Backend:** Supabase (Auth, Database, Storage)
- **Deployment Target:** Cloudflare Pages
